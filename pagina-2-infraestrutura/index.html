O que é Active Directory?

É um serviço de diretório da Microsoft que atua como uma banco de dados centralizado, executado no Windows Server, responsável por armazenar e gerenciar usuários, computadores, grupos e recursos de uma rede.

Principais funções e benefícios

  • Gerenciamento Centralizado: Administradores de T.I podem controlar toda a rede e os acessos a partir de um único local, sem precisar configurar cada computador individualmente.
